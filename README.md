## Dependencies
1. Install Mongodb from https://www.mongodb.com/try/download/community.
2. Nodejs 

### Running Nodejs App

1. `cd server`.
2. `npm install`
3. `node server.js`

### Running Angular app
1. `cd client`
2. `npm install`
3. `ng serve --port 8081`

Navigate to http://localhost:8081 and you are done
